# 使用说明

## 刷新串口按钮
点击后刷新当前电脑可用的串口，串口号会出现在右侧的下拉菜单中。

## 开始按钮
点击后会打开下拉菜单中选中的串口，开始收集数据并画图。

## 停止
点击后会停止所有操作

## 文件保存地址
用来保存文件的地址，默认为软件所在位置

## 开始保存按钮
点击后会分别保存三个通道的原始数据为txt文件，文件名为 "日期 + 时间 + data + 编号"。

## 结束保存按钮
点击后会停止本次的保存任务。
